## Deskripsi Tugas

Anda diberikan Struktur HTML dan CSS Eksternal untuk membuat laman web berdasarjan wireframe dan visual design. Tugas :

1. **Menambahkan Bootstrap classes** ke elemen HTML sesuai dengan visual design yang disediakan
2. **Membuat CSS styling** untuk mengatur layout, warna, dan tampilan sesuai wireframe
3. **Menambahkan sumber gambar** pada <img src="">.

### CSS Styling

Menambahkan styling untuk:

- Navbar
- Hero section
- Featured Categories
- Product Cards
- Buttons
- Footer
  Bebas menggunakan CSS variables yang sudah disediakan atau warna yang cocok (kreativitas):
- `--primary-green: #2ecc71`
- `--dark-text: #1a1a1a`
- `--light-gray: #f5f5f5`
- `--border-gray: #e0e0e0`
- `--text-muted: #999999`

### JavaScript

- Countdown timer sudah berfungsi/siap digunakan, silahkan implementasiskan ke html.

### Menambahkan Bootstrap Classes

- Contoh: Navbar
<nav class="navbar navbar-expand-lg navbar-light">
  <div class="container-fluid px-5">
    <!-- isi navbar -->
  </div>
</nav>
